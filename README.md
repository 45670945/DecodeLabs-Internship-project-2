

# 📊 Exploratory Data Analysis (EDA) Project

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on an e-commerce sales dataset.
The purpose of this analysis is to identify patterns, trends, customer behavior, and important business insights using statistical and analytical techniques.

The project was completed as part of the **DecodeLabs Data Analytics Industrial Training Program 2026**.

---

# 🎯 Project Objectives

* Analyze the dataset using descriptive statistics
* Identify trends and customer purchasing behavior
* Detect outliers in sales data
* Understand product performance
* Explore payment methods and referral sources
* Generate meaningful business insights

---

# 📂 Dataset Information

The dataset contains:

# 🔍 Key Findings

* Chairs generated the highest revenue.
* Instagram was the strongest referral source.
* Online payments were the most preferred payment method.
* Sales increased steadily over time.
* Several high-value outlier orders were detected.
* Cancelled and returned orders were relatively high.
